# Capability research status

Reference systems such as Tease AI, SexScripts/Groovy, Milovana EOS, and VirMst/CyberDom are capability research only. TeaseScript should reproduce useful capabilities through one modern deterministic engine; it does not run or copy their runtimes and formats.

Detailed historical comparisons and source archives are intentionally stored in the shared project research archive rather than this Git repository. A capability appearing there is not automatically accepted architecture, syntax, or release scope.
